## PwrightDemoBlaze
Automation Test for DemoBlaze website using Playwright - Typescript

### Project Structure

|__src
    |__data
    |__pages
    |__fixtures.ts
    |__helpers.ts
|__tests
    |__features
    |__regression
    |__performance

#### ⚡⚡⚡Running the Test
| Mode     | Command     |
|----------|:-----------:|
| Headless | npx playwright test  |
| Headed   | npx playwright test --headed |
| UI       | npx playwright --ui |
